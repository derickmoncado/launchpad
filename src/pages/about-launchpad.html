About Launchpad  

Launchpad is light-weight front end starter kit focused on automated web dev.
It's got eveything you need for super quick, static site development.

What baked in:

  • It's built on top of Boostrap 5
  • Powered by Gulp.js
  • Styled by Sass
  • Utilizes the standard 7-1 architecture pattern
  • Carousels from Swiper.js
  • Tooltips from Popper.js
  • Emergence.js for appending dynamic hooks on elements entering the viewport
  • Animate.css for sleek animations on elements (works great with Emergence.js)
  • HTML templating and partials with Panini
  • BrowserSync server + hot reloading on save
  • And even utilizes Boostrap Icons for extra flare

Launchpad also has...

  • Autoprefixing
  • Compilation
  • Sourcemaps
  • Concatenation
  • and Minification
