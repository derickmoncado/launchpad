How to Use Launchpad  

It's really easy, promise.

Launching dev environment

Well you already did since you're reading this. But just incase, it's:

$ gulp dev

This is your playground, build something cool, everything is served on port:3000



Production builds

When you're ready to ship, run:

$ gulp build

This will compiling, concatinate, and minify everything in to a folder called /dist
